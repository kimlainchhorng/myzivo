
# Phase 6: Mega Feature Update

## Feature 1: DMs Polish
- Read receipts (double-check marks)
- Typing indicators (animated dots)
- Message reactions (long-press emoji picker)
- Voice messages (record, play, waveform display)

## Feature 2: Explore/Discover Page
- User search with avatar previews
- Trending posts grid
- Hashtag browsing and filtering

## Feature 3: Profile Enhancements
- Post grid/list toggle views
- Follower/following lists with follow/unfollow actions
- Profile stats (posts count, followers, following)

## Feature 4: Reels/Video
- Short-form video recording with timer
- Vertical swipe playback feed with auto-scroll
- Like/comment/share overlay

## Feature 5: Travel Integration
- Flight search via Duffel API as shareable trip cards in feed
- Partner disclosure on checkout handoff

## Feature 6: Settings/Privacy Controls
- Block/unblock users
- Mute conversations
- Privacy toggles (profile visibility, activity status)

## Feature 7: Push Notifications
- Capacitor push notifications for messages, likes, follows
- Notification preferences in settings

## Feature 8: Saved/Bookmarks
- Save posts, flights, restaurants to collections
- Bookmarks page with categorized tabs
- DB table: `bookmarks` (user_id, item_type, item_id)

## Feature 9: Social Sharing
- Native share sheet via Capacitor/Web Share API
- Share posts, reels, profiles with deep links

## Feature 10: Map Integration
- Pin location on map for nearby users/restaurants
- Map view toggle on Explore page
