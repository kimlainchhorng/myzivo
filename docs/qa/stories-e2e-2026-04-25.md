## Storage-orphan validation query

After deleting a story (early or via expiry), run this in SQL editor; both should return `0`:

```sql
-- DB rows for expired stories
select count(*) from public.stories where expires_at < now();

-- Orphan storage objects (no matching story row, older than 1 day)
select count(*) from storage.objects o
where o.bucket_id = 'user-stories'
  and o.created_at < now() - interval '1 day'
  and not exists (
    select 1 from public.stories s
    where o.name like s.user_id::text || '/' || s.id::text || '/%'
       or o.name = split_part(s.media_url, '/user-stories/', 2)
       or o.name = split_part(s.audio_url, '/user-stories/', 2)
  );
```
